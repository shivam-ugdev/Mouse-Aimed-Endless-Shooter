# Mouse-Aimed Endless Shooter
An endless shooting game developed using Unity and C#.
The player can move, jump, aim using the mouse, and shoot enemies that fall from above. The objective is to eliminate enemies before they reach the ground. The project focuses on mouse-based aiming, shooting mechanics, enemy spawning, and endless gameplay logic.
This game was created as a learning project by following a Udemy tutorial.
## Features
- Player movement and jumping
- Mouse-based aiming
- Shooting mechanics
- Endless enemy spawning
- Falling enemy system
- Score-based gameplay
